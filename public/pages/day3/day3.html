Loops, Nodemon & Data

Loops is JavaScript

Immutability and a functional approach to loops are preferred as FOR loops are error prone, overly verbose and not functional. There's also no big difference in performance. Functional iterations don't modify the object.

Nodemon

Nodemon is CLI utility that wraps your Node app, watches the file system, and automatically restarts the process after every change. Source: digitalocean.com

It is a good idea to install it globaly as its avoids having to run the script after every change. Just executing command nodemon + app.js or index. js will run your project. You can also specify it in a script for the development environment.

It supports a configuration file -nodemon.json- where you can specify which file extentions to monitor for changes.

Sending Data

To Send data with GET requests, it has to be done through path variables OR query parameters/strings

PATH VARIABLES: url + /1

QUERY STRINGS: url + ?key=value&key2=value2
